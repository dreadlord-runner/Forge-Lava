This repository contains the website project inside the `forge-sample` folder.

1. Open a terminal in this repository.
2. Move into the project folder:

```powershell
cd forge-sample
```

3. Install dependencies:

```powershell
npm install
```

4. Start the development server:

```powershell
npm run dev
```

5. Open the local URL shown in the terminal.

Vite usually starts the site at:

```text
http://localhost:5173/
```
